# 📊 Customer Churn Analysis & Prediction

## 🚀 Overview
This project presents an end-to-end data analytics workflow focused on analyzing customer behavior and predicting churn in a telecom dataset.

The goal is to transform raw data into actionable insights that help businesses understand why customers leave and how to improve retention strategies.

---

## 🎯 Objectives
- Analyze customer usage patterns and behavior  
- Identify key factors influencing customer churn  
- Build machine learning models to predict churn  
- Create an interactive dashboard for decision-making  

---

## 🧰 Tools & Technologies
- **Python** (Pandas, NumPy)
- **Data Visualization** (Matplotlib, Seaborn)
- **Machine Learning** (Scikit-learn)
- **Business Intelligence** (Power BI)
- **SQL** (optional for data warehousing)

---

## 🧹 Data Preprocessing
- Handled missing values and removed duplicates  
- Standardized column formats  
- Encoded categorical variables for machine learning  
- Ensured data consistency and quality  

---

## 🔍 Exploratory Data Analysis (EDA)
- Generated summary statistics (mean, median, standard deviation)  
- Visualized distributions using histograms and boxplots  
- Identified relationships using scatter plots and correlation matrix  

---

## 🤖 Machine Learning

### ✔ Classification (Main Task)
- Models used:
  - Logistic Regression  
  - Random Forest  
  - Decision Tree  

- Evaluation metrics:
  - Accuracy  
  - Precision  
  - Recall  
  - F1-score  

### ✔ Clustering
- Applied K-Means clustering  
- Segmented customers based on usage patterns  

### ✔ Regression
- Built a linear regression model to analyze the relationship between usage and charges  

---

## 📊 Dashboard (Power BI)

An interactive dashboard was designed to provide a clear and executive-level view of customer behavior and churn drivers.
